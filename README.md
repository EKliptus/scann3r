# Scann3r

## Install

### Dependencies
    sudo apt-get install -y zip git nodejs npm pigpio redis-server imagemagick

### Setup Raspberry Camera

    sudo raspi-config

5 Interfacing Options => P1 Camera => Yes => Reboot

### Install Scann3r

    git clone https://github.com/sui77/scann3r.git
    cd scann3r
    npm install
